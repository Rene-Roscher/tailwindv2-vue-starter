# vue 2 + tailwind

Vue 2 CLI Starter with TailwindCSS v2

- Routing
- Page Examples
- Layout Example

## Added
- tailwindcss - 2.0.2
- @tailwindcss/forms - 0.2.1

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
